# Kiosk Project

## 개요

Kiosk Project는 무인 주문 시스템을 위한 CLI 기반 애플리케이션입니다. 사용자는 메뉴를 선택하고 장바구니에 담고 결제할 수 있습니다

## 주요 기능

- **사용자 모드**
  - 메뉴 선택 및 장바구니 기능
  - 결제 및 주문 확인

- **기타 기능**
  - 데이터 검증 및 예외 처리

## 기술 스택

- **언어**: Java
- **개발 도구**: IntelliJ IDEA
- **기타**: 파일 입출력, 컬렉션 프레임워크 활용
